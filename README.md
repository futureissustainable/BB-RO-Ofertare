# BioBuilds Offers

A web-based configurator and quote generator for BioBuilds' Passivhaus-certified modular homes.

## Features

- **Model Selection** - Configure 4 modular home models (24m² to 142m²)
- **Customization** - Choose finishes, facades, floor plans, and upgrades
- **Real-time Pricing** - Dynamic cost calculation based on selections
- **PDF Export** - Generate professional offers for customers
- **Multi-language** - Supports Romanian, English, German, and French

## Modes

- **View Mode** - Default display for customers
- **Edit Mode** - Add `#edit` to URL for sales team customization
- **Auxiliary Guide** - Add `#auxiliary` for supplementary pricing
